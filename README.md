# 🫀 Heart Disease Prediction

## 📌 Overview
This project uses **Machine Learning** techniques to predict whether a person has heart disease based on medical parameters. The dataset has been collected from **Kaggle**, and we experiment with multiple classification algorithms to achieve accurate predictions.

## 📊 Dataset
The dataset contains the following key features:
- **age**: Age of the person
- **sex**: Gender (1 = Male, 0 = Female)
- **cp**: Chest pain type (0-3)
- **trestbps**: Resting blood pressure (mm Hg)
- **chol**: Serum cholesterol (mg/dl)
- **fbs**: Fasting blood sugar (> 120 mg/dl, 1 = True, 0 = False)
- **restecg**: Resting electrocardiographic results (0-2)
- **thalach**: Maximum heart rate achieved
- **exang**: Exercise-induced angina (1 = Yes, 0 = No)
- **oldpeak**: ST depression induced by exercise relative to rest
- **slope**: Slope of the peak exercise ST segment
- **ca**: Number of major vessels (0-3) colored by fluoroscopy
- **thal**: Thalassemia type (0-3)
- **target**: 1 = Heart disease, 0 = No heart disease

## 🏆 Algorithms Used
We experiment with three different machine learning models:
1. **K-Nearest Neighbors (KNN)** 🏡
2. **Decision Tree Classifier** 🌳
3. **Random Forest Classifier** 🌲🌲
